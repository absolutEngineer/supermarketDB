This is a web application that aims at storing, managing and analysing the data gathered by a supermarket chain. Data about the stores, the customers, the transactions and the products are stored at the database. The web application offers the following functionalities:

  • Presentation of the data stored in the database using multiple filters
  • Presentation of the registered customers' profile and shopping habits
  • Statistical analysis of the transactions carried out in the supermarkets
  • Modification of the database entries (insert, edit or delete product, store and customer info)
  • Presentation of the price history for all products
